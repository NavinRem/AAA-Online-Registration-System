# AAA-Online-Registration-System

Full-stack online registration system using:

- Vue.js (Web)
- Flutter (Mobile)
- Firebase (Backend & Database)

## Project Structure

/web      - Web Application
/mobile   - Mobile App
/firebase - Firebase Config
/docs     - Documentation

## Setup

### Web
cd web
npm install
npm run dev

### Mobile
cd mobile
flutter pub get
flutter run

### Firebase
firebase login
firebase init
